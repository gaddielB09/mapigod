Proyecto de Mapas Web con Geolocalización

Este repositorio contiene un conjunto de 5 actividades prácticas enfocadas en el desarrollo de aplicaciones web con mapas, geolocalización del usuario y persistencia de datos.
Cada actividad se encuentra organizada en su propia carpeta y representa una etapa incremental del proyecto.

La Actividad 5 integra todas las funcionalidades desarrolladas en las actividades anteriores y fue desplegada utilizando PythonAnywhere.


Objetivo del proyecto

-El objetivo principal es comprender y aplicar:
-Geolocalización en aplicaciones web
-Uso de mapas interactivos
-Manejo de eventos en el mapa
-Persistencia y visualización de datos
-Integración completa de funcionalidades en una sola aplicación

Despliegue

La aplicación final (Actividad 5) fue desplegada utilizando PythonAnywhere, permitiendo su acceso desde la web sin necesidad de ejecución local.

El despliegue no afecta la estructura del repositorio, únicamente la ejecución del proyecto final.
